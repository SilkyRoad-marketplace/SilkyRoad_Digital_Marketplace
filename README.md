# Silky Road Digital Marketplace

A static HTML-based digital marketplace for buying and selling ebooks, art, and music.

## Pages
- `index.html` – homepage with featured products and categories
- `contact.html` – contact form (connected to Supabase)
- `seller.html` – become a seller page
- `signup.html` – signup form connected to Supabase
- `dashboard.html` – seller dashboard to track sales

## Tech
- HTML, CSS, JavaScript
- Supabase (database, auth, file storage)
- PayPal API (payout system)
- Hosted on Vercel (static site mode)
